# API Endpoints Documentation

## Description
This file contains the documentation for the API endpoints of the ecommerce website for drawings.

## Endpoints
- GET /api/drawings
- POST /api/drawings
- PUT /api/drawings/:id
- DELETE /api/drawings/:id

## Sample Request
```json
{
  "title": "New Drawing",
  "artist": "Artist Name",
  "price": 100
}
```
